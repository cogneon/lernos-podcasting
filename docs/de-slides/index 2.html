Kurz zu lernOS

  • Offenes System für Lebenslanges Lernen und Lernende Organisationen
  • lernOS ist ein Verb in Esperanto, Ich/wir werde/werden lernen
  • Beratungsinhalte der Cogneon Akademie als Open Educational Resource
  • Leitfäden sind auf lernos.org, auf Github unter Creative Commons Lizenz (CC BY 4.0) verfügbar

Geschichte des Leitfadens

  • Juni 2017: Inhalt von Tim’s pb21-Dokumente (CC BY) in Wikibook übernommen und Podcasting-Workflow-Grafik ergänzt
  • Oktober 2017: Wikibook in lernOS Format übernommen (auf Subscribe 9 in München)
  • Juli 2019: Teile der Sketchnote Podcast in a Nutshell von Marco übernommen
  • Dezember 2019: Grafik Podcasting Canvas ergänzt (auf 36c3)

Verfügbare Leitfaden-Versionen

  • Markdown - unser Quellformat
  • Webversion - responsives Web-Interface
  • PDF - zum ausdrucken/annotieren (pandoc, TeX)
  • Word - für einfache Anpassungen (pandoc)
  • HTML - rohes HTML für copy&paste (pandoc)
  • E-Book - Formate mobi&epub (pandoc, ImageMagick, Calibre)


Ideen für die Zukunft: Audiobook- und OneNote-Version, automatische Übersetzung, Intranet-Beispiele, weitere Quellen z.B. 24 Tipps zum Podcasten, Video-Lernpfad
